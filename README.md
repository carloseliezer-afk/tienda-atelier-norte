# Tienda de ropa web

Proyecto web estatico de una tienda de ropa hecho con HTML, CSS y JavaScript.

## Archivos

- `index.html`: estructura principal de la tienda
- `styles.css`: estilos y diseno responsive
- `app.js`: productos, filtros, carrito y logica interactiva

## Como ejecutar

1. Abre esta carpeta en Visual Studio Code o Visual Studio.
2. Abre `index.html`.
3. Ejecuta con Live Server o abre el archivo directamente en tu navegador.

## Funciones incluidas

- Catalogo dinamico con 9 productos de ejemplo
- Filtros por categoria, estilo, busqueda y orden
- Carrito funcional con almacenamiento en `localStorage`
- Resumen de compra y simulacion de checkout
- Diseno adaptable para escritorio y movil

## Personalizacion rapida

- Edita el arreglo `products` en `app.js` para cambiar prendas, precios o categorias.
- Ajusta colores y estilos desde `:root` en `styles.css`.
- Modifica los textos principales en `index.html`.
